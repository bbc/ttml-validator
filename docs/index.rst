.. ttml-validator documentation master file

Welcome to ttml-validator's documentation!
==========================================

ttml-validator is a Python project for validating TTML_ files.
It currently validates two profiles, EBU-TT-D_ and `IMSC Text`_
conforming to the `BBC Subtitle Guidelines`_, and DAPT_.
